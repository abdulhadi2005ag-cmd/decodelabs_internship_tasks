# To-Do List CLI

A simple command line To Do List application built in Python as Task 1 of my Python Programming internship at DecodeLabs.

## What it does

- Add tasks to a list
- View all tasks with numbering
- Exit the program

## How to run

```bash
python todo_list.py
```

Then follow the menu:
1. Add task
2. View tasks
3. Exit

## What I learned

- Storing multiple items in a list using `.append()`
- Looping through a list to display its contents
- Building a simple menu driven program with a while loop

## Tech used

- Python 3
